Nodebots

We can use node.js, and all the stuff we're learning in class, to program little robot-like computers. This is what node-bots is all about.

johnny-five

johnny-five is the thing that bridges the gap between node.js and the devices that operate in the physical world, like Arduinos and Raspberry Pis. It's a node module, so you can install it and require it just like fs or express.

Hello world

If you have an Arduino, follow these instructions to get your board connected. If you don't have a board, no problem, you can still get started with nodebot-workshop...

nodebot-workshop

nodebot-workshop is series of easy code challenges that teach you how to write node.js for the Arduino. Do a global install and then run it from the command line. It works whether you have a board connected or not.
